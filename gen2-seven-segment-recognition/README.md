## Install requirements

```
python3 -m pip install -r requirements.txt
```

## Download blobs:
```
python3 download.py
```

## Run model

```
python3 main.py -nn blobs/7_SEG_256x750_MONO_simplified_openvino_2021.4_6shave.blob
```
